Quick openvpn client setup guide


  1. Make sure your source list is up-to-date.
    apt-get update
  2. Install openvpn.
    apt-get install openvpn
  3. Untar downloaded .tar file. Move following extracted files to /etc/openvpn
    • ca.crt
    • user_name.key
    • user_name.crt
    • client.conf
  4. Start openvpn service.
    service openvpn start
  5. You will be prompted for the username and passwd. Enter your LDAP(GCL) credentials. On successful connection, client will be automatically assigned an IP addess in the range 10.8.0.x with routes set to the Baadal internal VLANs.
